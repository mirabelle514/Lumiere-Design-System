Usage Guidelines

Colors: Use Parisian Navy for primary text, Antique Gold for borders only (never text), and Ivory for backgrounds.

Typography: Playfair Display for headings creates elegance, Open Sans ensures readability.

Spacing: Use the 8px grid system for consistent spacing throughout.
